Project Overview

Pizza Sales Analysis using SQL

This project analyzes pizza sales data using SQL to uncover
business insights such as revenue generation, customer ordering
patterns, best-selling pizzas, and sales performance.

-------------------------------------------------------
Business Questions Solved
1. Total Revenue Generated
2. Average Order Value
3. Most Ordered Pizza
4. Top Revenue Generating Pizzas
5. Sales Distribution by Category
6. Highest Value Orders
7. Pizza Size Analysis
